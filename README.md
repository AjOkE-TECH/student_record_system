# Student Record Management System

## Project Description

A web-based system for managing student records.

## Features

* Admin Login
* Add Student
* View Students
* Edit Student
* Delete Student
* Search Student

## Technologies Used

* PHP
* MySQL
* CSS

## Installation

1. Clone the repository
2. Move project to XAMPP htdocs
3. Import database
4. Start Apache and MySQL
5. Open localhost/project-folder
